🎯 이 대시보드는

Qoo10 JP에서 한국·일본 경쟁 뷰티 브랜드가 지난 3주간 출시한 신상품을 자동 수집해 보여줍니다.

  • 대상: 자사 4브랜드(BOH·브링그린·웨이크메이크·컬러그램) 경쟁사 56개
  • 갱신: 매일 09:00 JST 자동 (GitHub Actions)
  • 목적: 상품기획팀이 "다음에 뭘 만들어야 하나" 판단

📝 "신상품"으로 잡는 기준

두 조건 모두 만족하는 것:

  1. Qoo10 상품명에 신상품 마커 (【NEW】 新発売 新作 新色 리뉴얼 先行発売 등)
  2. 첫 관측일로부터 21일 이내 (그 이후는 "졸업")

셀러가 NEW 마커 떼면 다음 크롤에서 자동 제외. 21일 지나면 신선도 cutoff로 자동 졸업.

📊 상단 요약 (스킨 / 색조)

카테고리별 한눈에 보는 숫자:

  • 신상품 수 / 브랜드 수 — 출시 활성도
  • 평균 판매가 — 가격대 정합성 점검용
  • 평균 할인율 — 30%+ = 메가와리 임박 신호
  • 세부 카테고리 평균가:
    • 스킨: 크림 / 세럼 / 토너 / 마스크 / 패드 별
    • 색조: 립 / 아이 / 베이스 / 페이스 별 + 세부 Top 5

🎯 다음 상품 방향성 (가장 중요)

제품명 텍스트를 자동 분석해 "우리는 다음에 뭘 만들지"를 제안:

  • 스킨케어 탭 — 핫 성분 (PDRN·콜라겐 등) / 핫 제형 / 핫 효능
  • 색조 탭 — 핫 카테고리(립틴트 등) / 핫 제형 / 핫 색계열 / 호수 패턴
  • 자사 액션 ⚡ 탭 — BOH/브링그린/웨이크/컬그용 구체 액션 7개

셀러가 NEW로 푸시 중 = 이미 마케팅 강도가 높은 상품. 트렌드 신호 신뢰도 높음.

📋 테이블 컬럼 의미

  • 구분 — 신제품 신규 색상 리뉴얼
  • Qoo10 # — 뷰티 베스트 Top 200 순위 = 구매 검증
  • LIPS ★ — LIPS 카테고리 랭킹 = SNS 화제도 (10s~30s)
  • @cosme — 주간 Top 10 = 평점 권위 (30s~50s)
  • 리뷰 — Qoo10 카드의 별점·개수
  • 시그널 — 어떤 신상품 마커 매칭됐는지
  • 브랜드 셀의 작은 라벨 — Nd째 첫 관측 후 경과일

세 차원 동시 진입 SKU = 자사 차기 기획 1순위 벤치마크.

📥 시트로 옮기기

결과 표 오른쪽 위 📋 시트로 복사 (TSV) 버튼:

  1. 현재 필터 결과를 TSV(탭 구분) 형식으로 클립보드 복사
  2. 구글 시트 "타사 신제품" 워크북 열고 셀에 붙여넣기
  3. 컬럼이 시트 양식 그대로 정렬되어 들어감

⚠ 알아둘 한계

  • 등록일은 추정 — 정확한 Qoo10 등록일 fetch는 차단됨. 셀러가 NEW 떼는 시점 + 첫 관측일로 보완
  • Qoo10 미입점 brand는 빠짐 — Round Lab, PYUNKANG YUL, 일본 디바이스 등은 Rakuten/Amazon 주력
  • @cosme 매칭이 적음 — Top 10만 가져오므로 매칭 풀 작음. 운영 누적되면 증가
  • 약한 마커 오탐 — 제품명에 "NEW"만 들어간 케이스는 신상이 아닐 수도 있음
